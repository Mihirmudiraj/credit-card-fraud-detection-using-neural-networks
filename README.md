# credit-card-fraud-detection

End-to-end deep learning project for detecting fraudulent credit card transactions using TensorFlow/Keras. Includes data cleaning, EDA, feature engineering, neural network training, and evaluation with precision, recall, F1 score, and confusion matrix. Demonstrates handling of highly imbalanced financial fraud datasets.

## Project Overview

This project demonstrates an end-to-end deep learning pipeline for detecting fraudulent credit card transactions.  
The workflow includes data cleaning, exploratory data analysis (EDA), feature engineering, neural network training using TensorFlow/Keras, and model evaluation using classification metrics.

Fraud detection is an important real-world financial problem where identifying suspicious transactions early can help prevent financial loss.

## Dataset

The project uses the **Credit Card Fraud Detection dataset** from Kaggle.

Dataset source: [Kaggle Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)

Key characteristics:
* Highly imbalanced dataset
* Fraud cases are very rare compared to normal transactions
* Most features are anonymized using PCA transformation
* Includes transaction time and amount

2. Exploratory Data Analysis (EDA)

## Model Evaluation

The trained model is evaluated using:
* Accuracy  
* Precision  
* Recall  
* F1 Score  
* Confusion Matrix  

These metrics help evaluate performance on an imbalanced fraud detection dataset.
